# 🚀 MarkGrow - Digital Marketing Agency Website

MarkGrow is a modern and responsive digital marketing agency website designed to help businesses grow their online presence and brand value.
This project showcases services like product marketing, brand growth, social media marketing, PR strategies, and digital advertising.

---

## 🌐 Live Demo

https://shirishh31.github.io/MarkGrow-Project/

---

## 📌 Features

* ✨ Modern and clean UI design
* 📱 Fully responsive (mobile, tablet, desktop)
* 📢 Digital marketing service showcase
* 📊 Brand growth and strategy sections
* 💼 Product selling & promotion sections
* 📣 Social media marketing highlights
* 🎯 Digital advertising solutions
* 📞 Contact form with user interaction
* 🗺️ Integrated Google Maps for location
* 🔗 Social media links (Instagram, LinkedIn)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 🧠 Project Purpose

This project is created to demonstrate front-end development skills and build a real-world digital marketing agency website.
It can be used as a template for agencies, freelancers, or startups looking to establish an online presence.
